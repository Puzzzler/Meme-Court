# Current Sprint
1

## Goal

Automatically react to memes with thumbs-up and thumbs-down votes.

## Success Criteria

- [x] Bot ignores itself.
- [x] Bot only reacts in the meme channel.
- [x] Adds exactly two reactions.
- [x] Doesn't crash if it can't react.

## Questions

- Should GIFs count? No
- Should image uploads count? Yes
- Should text-only memes count? No

## Notes

- Uploaded image attachments posted in `meme-court` receive voting reactions.
- Text-only posts, GIF uploads, and image links without attachments do not receive voting reactions.
